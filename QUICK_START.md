### الحل السريع:

1. **افتح لوحة الإدارة:**
   ```
   http://localhost:3000/admin-panel.html
   ```

2. **ابحث عن المستخدم الموظف** في القائمة

3. **اضغط على "إعادة تعيين كلمة المرور"**

4. **أدخل كلمة مرور جديدة** (مثلاً: `123456`)

5. **سجل دخول بالبريد الإلكتروني وكلمة المرور الجديدة**

## خطوات الاستخدام:

### 1. تسجيل دخول كمدير
```
http://localhost:3000
Email: mostafa.nassar@4am-system.com
Password: Admin@123
```

### 2. إرسال تذكرة للموظف
- اذهب إلى "التذاكر"
- اضغط "إرسال تذكرة جديدة"
- اختر الموظف
- اكتب الرسالة
- اضغط "إرسال"

### 3. تسجيل دخول كموظف
- سجل خروج
- سجل دخول بحساب الموظف
- ستجد التذكرة في صفحة "التذاكر"

### 4. إضافة حافز/خصم
- سجل دخول كمدير
- اذهب إلى "الحوافز والخصومات"
- اضغط "إضافة حافز/خصم"
- املأ البيانات
- اضغط "حفظ"

### 5. عرض الحوافز (كموظف)
- سجل دخول كموظف
- اذهب إلى "حوافزي"
- ستجد جميع الحوافز والخصومات الخاصة بك

## حل المشاكل الشائعة:

### المشكلة: "Access denied"
**الحل:** تأكد من تسجيل الدخول أولاً

### المشكلة: "السيرفر غير شغال"
**الحل:** شغل السيرفر بالأمر `npm start`

### المشكلة: "نسيت كلمة المرور"
**الحل:** استخدم لوحة الإدارة لإعادة تعيين كلمة المرور

### المشكلة: "البيانات اختفت بعد إعادة تشغيل السيرفر"
**الحل:** البيانات الآن محفوظة في ملف `database.json` ولن تختفي
