# تشغيل السيرفر

## المشكلة الحالية
الخطأ الذي يظهر `Unexpected token '<' "<!DOCTYPE"` يعني أن السيرفر غير شغال.

## الحل

### 1. افتح Terminal جديد في المجلد الحالي

### 2. شغل السيرفر بأحد الأوامر التالية:

**الطريقة الأولى (الأساسية):**
```bash
npm start
```

**الطريقة الثانية (للتطوير مع إعادة التشغيل التلقائي):**
```bash
npm run dev
```

### 3. انتظر حتى تظهر رسالة:
```
Server running on http://localhost:3000
```

### 4. الآن افتح المتصفح على:
```
http://localhost:3000/register.html
```

## ملاحظات مهمة:
- يجب أن يبقى الـ Terminal مفتوح طول فترة استخدام النظام
- إذا أغلقت الـ Terminal، السيرفر سيتوقف
- إذا حدث خطأ، تأكد من تثبيت المكتبات أولاً بالأمر: `npm install`

## التحقق من أن السيرفر شغال:
افتح المتصفح على: http://localhost:3000
إذا ظهرت صفحة تسجيل الدخول، معناها السيرفر شغال بنجاح ✅
